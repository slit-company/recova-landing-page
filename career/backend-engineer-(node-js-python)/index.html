About the Role


As a Backend Engineer at Recova, you will be responsible for designing and developing the core systems, APIs, and services that power our platform. You’ll work with a modern stack, ensure high scalability, and build reliable backend solutions that support thousands of users.
This role is ideal for engineers who love solving deep technical challenges, value clean architecture, and can collaborate effectively across teams.


  • Build and maintain scalable backend services using Node.js or Python

  • Design RESTful and GraphQL APIs for frontend and mobile applications

  • Implement secure authentication, authorization, and data workflows

  • Optimize databases, queries, and overall system performance

  • Collaborate closely with frontend, product, and DevOps teams

  • Ensure best practices in documentation, testing, and code structure


Responsibilities


  • Develop high-performance backend features and microservices

  • Architect, build, and maintain database schemas (PostgreSQL, MongoDB, Redis)

  • Integrate third-party APIs and internal services

  • Improve system reliability, uptime, and scalability

  • Write unit/integration tests and contribute to QA automation

  • Participate in sprint planning, code reviews, and architecture meetings

  • Monitor performance, logs, and production health with DevOps tools

  • Implement security best practices (encryption, rate limiting, token handling)


Requirements / Qualifications


  • 3–6+ years of experience in backend development

  • Strong proficiency in Node.js, Express.js and/or Python (FastAPI, Django, Flask)

  • Solid understanding of relational and NoSQL databases

  • Experience building REST/GraphQL APIs

  • Familiarity with Docker, CI/CD, and cloud platforms (AWS preferred)

  • Ability to write clean, scalable, and testable backend code

  • Strong problem-solving, debugging, and architecture skills

  • Experience in SaaS, CRM, workflow tools, or high-scale systems (plus point)

  • Comfortable working in an agile, fast-paced environment


Benefits and Culture Highlights


  • Attractive salary + annual performance bonus

  • Remote-friendly flexible work culture

  • Health, wellness, and mental-health support

  • Paid holidays and flexible PTO

  • Learning budget for courses, certifications, and events

  • Opportunity to work with modern backend tools & cloud technologies

  • Collaborative, engineering-driven culture that values innovation

  • Strong growth path toward Senior/Lead Backend Engineer roles
